Updater

RMS can check for and apply updates automatically from GitHub.

How It Works

📡

Version Check

RMS fetches the latest version number from GitHub and compares it to your installed version.

⬇️

Download

If an update is available, RMS downloads only the changed files from the latest GitHub release.

🔁

Apply

Files are overwritten in your %AppData%\RMS\dist\ folder. A Discord restart applies the update.

Checking for Updates Manually

1

Open Discord Settings → RMS → Updater

Scroll to the RMS section in settings and click Updater.

2

Click Check for Updates

RMS will check GitHub for the latest version and tell you if you're up to date or not.

3

Click Update if available

If a new version is found, click the Update button to download and install it.

4

Restart Discord

After updating, fully quit and reopen Discord to apply the changes.

Auto Update

You can enable auto-update in the Updater tab. When enabled, RMS will automatically check and apply updates every time Discord starts, with no manual steps needed.

ℹ️ Only files attached to the GitHub release are updated. Your settings, themes, and QuickCSS are never touched by the updater.
