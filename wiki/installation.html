Installation

Follow these steps to install RMS on your PC.

⬇ Download RMSInstaller.zip

Requirements

  • Windows 10 or later
  • Discord desktop app installed
  • Internet connection (installer downloads files from GitHub)

Install Steps

1

Download the zip

Click the download button above to get RMSInstaller.zip.

2

Extract the zip

Right-click the zip → Extract All. Open the extracted folder.

3

Unblock the exe

Right-click RMSInstaller.exe → Properties → check Unblock at the bottom → click OK. This stops Windows from blocking it.

4

Run the installer

Double-click RMSInstaller.exe. If SmartScreen appears, click More info → Run anyway.

5

Accept the license

Read and accept the license agreement, then click Continue.

6

Click Install

The installer will download the latest RMS files from GitHub and patch Discord automatically.

7

Restart Discord

Fully quit Discord (right-click tray icon → Quit Discord) and reopen it. You should now see RMS Settings in your Discord settings.

If Windows Blocks It

⚠️ Windows SmartScreen may block the installer because it's not signed by a major certificate authority. This is normal for open-source tools.

Option 1 — Unblock in Properties

1

Right-click the exe

Select Properties from the menu.

2

Check Unblock

At the bottom of the General tab, check the Unblock checkbox.

3

Click OK and run

Click OK then double-click the exe again.

Option 2 — SmartScreen popup

If SmartScreen shows a blue popup saying "Windows protected your PC", click More info then Run anyway.

Option 3 — PowerShell

Open PowerShell and run:

Unblock-File -Path "C:\path\to\RMSInstaller.exe"

Uninstall

Open RMSInstaller.exe again and click Uninstall. This will remove all RMS files and restore Discord to its original state. Then restart Discord.
