What is RMS?

RMS is a modification for Discord's desktop app. It lets you customize your Discord experience with plugins that add new features and themes that change how it looks — all without touching Discord's official files in any unsafe way.

RMS is built on top of Vencord and maintained by zxkuhl. It is not affiliated with or endorsed by Discord Inc.

Features

🧩

Plugins

Enable powerful mods like FakeNitro, NoTrack, and more with one click.

🎨

Themes

Load CSS themes from a URL or write your own with QuickCSS.

🔄

Auto Updater

Stay up to date automatically. RMS checks for updates on startup.

☁️

Cloud Sync

Back up and restore your settings across devices with cloud sync.

🧪

Docs Tab

Built-in documentation right inside Discord's settings panel.

🔒

No Tracking

NoTrack plugin disables Discord's analytics by default.

Quick Start

1

Download the Installer

Click the Download button above to get RMSInstaller.zip.

2

Extract and Run

Extract the zip, right-click RMSInstaller.exe → Properties → Unblock, then run it.

3

Accept & Install

Accept the license agreement and click Install. RMS will download and set itself up automatically.

4

Restart Discord

Fully quit Discord and reopen it. RMS Settings will appear in your Discord settings panel.

⚠️ RMS modifies Discord's client. Using it may violate Discord's Terms of Service. Use at your own risk.
